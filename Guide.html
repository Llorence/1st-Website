Welcome! This page will assist you in successfully downloading and installing the map


1. Download the file

2. Locate your Terraria Maps folder, it should be located

3.Users\*username*\Documents\My Games\Terraria\Worlds

4. Extract "Super Mario Terraria - The Series" into the worlds folder

5. Launch the game, in Terraria you should see both of the Super Mario Terraria maps in your map selection.

6. DONE!

That should get you set up correctly, if you still have a problem, contact us, good luck!
